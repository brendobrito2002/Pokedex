Requisição HTTP

URL: https://pokeapi.co/api/v2/pokemon/1

${Endereço}/${path = caminho de identificação do recurso}${ID}

Endereço: https://pokeapi.co

IP: https://127.0.0.1:8080

Path: /api/v2/pokemon

ID: Identificação do recurso

O que fazer com o recurso:

Request Method

  • GET: Pegar;
  • POST: Inserir;
  • PUT: Atualizar;
  • DELETE: Deletar;

URL: https://pokeapi.co/api/v2/pokemon?type=Grass

Querry String(GET): Tudo que vem depois da '?', filtro de estrutura chave=valor, mais de um parametro utilizar o '&'chave=valor

Exemplo: https://pokeapi.co/api/v2/pokemon?offset=151&limit=102

Neste caso, está sendo requesitado os pokemons da segunda geração

offset= a partir do número

limit= número máximo

Request/Response Header

Configuração da API

Exemplo Request:

Accept-Language: pt-BR,pt;q=0.9,en-US;q=0.8,en;q=0.7

Exemplo Response:

Content-Type:application/json; charset=utf-8

Request Body(POST)

O corpo do conteúdo, é o 'content-type' retornado pelo response header

Body{ "name":"Teste" }

Status Code

O que aconteceu com a requisição

200 - 299: OK

300 - 399: Redirected

400 - 499: ERRO por parte do cliente

500 - 599: ERRO interno do servidor
